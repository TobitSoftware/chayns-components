<h2 style={{ marginTop: '32px' }}>How to use</h2>

<p style={{ fontSize: '1.1rem' }}>
    Initially, ensure the installation of the core package along with any additional packages you wish to incorporate.
    Follow the commands outlined in the <b>Installation</b> guide for this purpose.
    Once all the packages are successfully installed, enclose your top-level
    elements with the ColorSchemeProvider as demonstrated below:
</p>

```tsx
import React from 'react';
import { ColorSchemeProvider } from '@chayns-components/core';
import { ChaynsProvider, getSite } from 'chayns-api';

const AppWrapper = () => {
    const { color, colorMode } = getSite();

    return (
        <ChaynsProvider>
            <ColorSchemeProvider color={color} colorMode={colorMode}>
                <YourComponent/>
            </ColorSchemeProvider>
        </ChaynsProvider>
    )
}

export default AppWrapper;
```

<h2 style={{ marginTop: '32px' }}>Contribution</h2>

<p style={{ fontSize: '1.1rem' }}>
    Make sure that your node version is 16 or above.
    To begin, install all the dependencies for the main project and its sub-projects,
    and then link them. Simply use the provided commands:
</p>

```bash
# NPM
npm install

# Yarn
yarn install
```

<p style={{ fontSize: '1.1rem' }}>
    Next, run one of the given commands to initiate the local version of Storybook and start developing within it:
</p>

```bash
# NPM
npm run storybook

# Yarn
yarn run storybook
```

> **Information:** The local Storybook can be accessed through port 6006 on your localhost.
> You can visit it using the following URL: `http://localhost:6006/`
